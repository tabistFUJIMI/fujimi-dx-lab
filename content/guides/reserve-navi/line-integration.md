## このガイドを読むとこうなります

| Before | After |
|---|---|
| ネット予約はできるが、LINEからの問い合わせは手動対応 | LINEトークで「予約する」ボタンから直接予約完了 |
| 予約確認メールが迷惑メールに埋もれる | LINE通知で確実に届く |
| 顧客の本名が分かりづらい（LINE表示名のみ） | 予約時に入力した情報がカルテに自動紐付く |

## 前提条件

- LINE公式アカウントを取得済み（[LINE Official Account Manager](https://www.linebiz.com/jp/)）
- Messaging API が有効化されている
- Reserve Navi が「ライト」プラン以上で運用中

:::warning
フリープランでは LINE 連携は使えません。ライト（¥980/月）以上にアップグレードが必要です。
:::

---

## Step 1: LINE側で必要な情報を取得

### 1-1. チャネルアクセストークンの発行

1. [LINE Developers Console](https://developers.line.biz/) にログイン
2. 対象のチャネルを選択
3. 「Messaging API設定」タブ
4. **チャネルアクセストークン（長期）** → 「発行」
5. 発行されたトークンをコピー（後で使う）

:::warning
このトークンは**パスワード相当の機密情報**です。他人に見せない、メール等で送らない。
:::

### 1-2. LIFF ID の発行（ミニアプリ用）

1. 同画面の「LIFF」タブ
2. 「追加」で新規LIFF作成:

| 項目 | 値 |
|---|---|
| LIFFアプリ名 | 予約ページ |
| サイズ | Full |
| エンドポイントURL | `https://reserve-navi.fujimin-pass.com/book/<あなたの店舗スラッグ>` |
| Scope | profile, openid |
| ボットリンク機能 | On（Aggressive） |

3. 作成後に表示される **LIFF ID** をコピー

---

## Step 2: Reserve Navi 側で接続設定

1. Reserve Navi 管理画面 → 「設定」→「LINE連携」
2. 以下を入力:

| 項目 | 入力内容 |
|---|---|
| チャネルアクセストークン | Step 1-1 で取得したトークン |
| LIFF ID | Step 1-2 で取得したID |

3. 「保存して接続テスト」をクリック

:::success
「LINE API 接続成功」と表示されれば Step 3 へ。
:::

### ❓ 接続失敗したとき

| 症状 | 原因 | 対処 |
|---|---|---|
| 「Invalid token」 | トークン誤り／期限切れ | LINE Developers Console で再発行 |
| 「LIFF not found」 | LIFF ID誤り | IDをコピーし直す |
| タイムアウト | LINE側障害 | 15分待って再試行 |

---

## Step 3: リッチメニューを設定（任意だが強く推奨）

リッチメニューがあると、お客様は **トーク画面の下部から1タップで予約ページに飛べる**ようになります。

### 自動設定を使う場合

1. Reserve Navi 管理画面 → 「設定」→「LINE連携」→「リッチメニューを自動設定」
2. テンプレート（予約する／お問い合わせ／営業時間）から選択
3. 「適用」

:::info
Reserve Navi があらかじめ用意しているテンプレート画像を使うので、デザイン不要で完成します。
:::

### 独自メニューを使う場合

LINE Official Account Manager の「リッチメニュー」から手動設定も可能。予約ボタンのリンク先は:

```
https://liff.line.me/<あなたのLIFF ID>
```

---

## Step 4: Webhook の確認

1. LINE Developers Console → 「Messaging API設定」
2. Webhook URL が自動的に:

```
https://reserve-navi.fujimin-pass.com/api/webhook/line
```

になっていることを確認

3. 「Webhookの利用」が **オン** になっていることを確認
4. 「応答メッセージ」が **オフ** になっていることを確認（Reserve Navi 側で制御するため）

---

## Step 5: 動作テスト

1. LINEで自店舗の公式アカウントを友だち追加
2. リッチメニューから「予約する」→ LIFF予約ページが開く
3. 好きなメニュー・日時を選択 → 予約送信
4. Reserve Navi 管理画面に予約が即座に反映される
5. LINEトークに予約完了メッセージが届く

:::success
ここまで通れば LINE連携完了。お客様の使い勝手を想像しながら、自分で一度予約の流れを試すと安心です。
:::

---

## 🎉 導入効果の目安

運用3ヶ月の店舗データ：

- **LINE経由予約比率**: 30-50%（開始1ヶ月で急増）
- **予約→来店率**: 85-95%（リマインドLINEで当日キャンセル率低下）
- **無断キャンセル**: 5% → 1% 程度まで削減

## 運用のコツ

- リッチメニューに「予約する／キャンセル／お問い合わせ／営業時間」の4つを置くのが王道
- 予約完了後、お礼＋次回おすすめメニューをLINEで送ると再来店率UP
- ブロックされにくくするため、LINE配信は**月1〜2通**まで
- 予約確認用の自動応答は短く、フレンドリーに
