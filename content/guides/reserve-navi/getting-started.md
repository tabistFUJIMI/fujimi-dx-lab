## このガイドを読むとこうなります

| Before | After |
|---|---|
| 電話や手書き台帳で予約を管理していて、ダブルブッキングが発生 | スマホ1台で予約を一元管理、ダブルブッキングゼロ |
| お客様から「予約したいけど営業時間外でつながらない」 | ネット予約で24時間受付、店舗閉店後も予約が入る |
| 「顧客の過去来店履歴」がバラバラで常連管理が難しい | カルテ機能で来店回数・施術内容・好みを一画面で確認 |

## 所要時間・レベル

- **時間**: 最短15分（基本設定）／ 実運用開始まで30分
- **レベル**: ★☆☆（PC/スマホでフォーム入力ができればOK）
- **必要なもの**: メールアドレス、店舗名、営業時間、提供メニュー一覧

---

## Step 1: アカウント開設

FUJIMIN PASS の新規登録から開始します（既に登録済みの方は Step 2 へ）。

1. [FUJIMIN PASS 新規登録](https://www.fujimin-pass.com/register) にアクセス
2. メールアドレス・パスワード・店舗名・代表者名を入力
3. 確認メールのリンクをクリックしてアカウント有効化

:::success
登録完了後、FUJIMIN PASS 管理画面にログインできれば成功です。
:::

---

## Step 2: Reserve Navi を有効化

1. FUJIMIN PASS 管理画面 → 「サービス一覧」
2. **Reserve Navi** のカードで「始める」
3. プラン選択（フリー / ライト / スタンダード / プロ）

:::info
フリープランで月70件までの予約管理を無料で試せます。クレカ登録不要。
:::

---

## Step 3: 店舗情報の登録

管理画面の **設定 → 店舗情報** で以下を入力します。

| 項目 | 例 |
|---|---|
| 店舗名 | Relaxia 整体院 |
| 店舗スラッグ | relaxia （URL `/book/relaxia` の一部になる英数字） |
| 住所・電話番号 | 予約完了メール・地図表示に使う |
| 営業時間 | 10:00〜21:00（最終受付 20:30） |
| 定休日 | 水曜 |

:::warning
**店舗スラッグは変更できません。** 短く覚えやすく、お店の名前に近いものを選んでください。後からブックマーク・看板QRコードを作ると、変更はほぼ不可能になります。
:::

---

## Step 4: メニューの登録

**設定 → メニュー** で、お客様が予約時に選ぶメニューを登録します。

### 入力項目

| 項目 | 例 |
|---|---|
| メニュー名 | 全身ほぐし 60分 |
| 所要時間 | 60分 |
| 料金 | ¥6,600（税込） |
| 説明（任意） | デスクワーク疲れに人気のコース |

:::tip
最初は **主力メニュー 3〜5件** を登録すれば十分。完璧を目指さず、運用しながら追加していくのが現場では王道です。
:::

---

## Step 5: スタッフの登録

1人運営の場合も「代表者」をスタッフとして登録する必要があります。

**設定 → スタッフ** で以下を設定:

- 氏名
- 対応可能メニュー（複数選択可）
- 勤務時間（営業時間と異なる場合）

:::info
スタッフ複数の場合、お客様は予約時に指名/おまかせを選べます。指名料の追加設定も可能。
:::

---

## Step 6: ネット予約ページの公開

設定が完了したら、以下のURLにアクセスして予約ページの動作を確認しましょう。

```
https://reserve-navi.fujimin-pass.com/book/<あなたの店舗スラッグ>
```

例: スラッグが `relaxia` なら `https://reserve-navi.fujimin-pass.com/book/relaxia`

:::success
お客様視点で、メニューを選び → 日時を選び → 氏名・電話を入力 → 予約完了 が通れば運用OK。
:::

### 集客導線の作り方

- 店頭やレシートに **予約URLをQRコード化して掲示**
- Google ビジネスプロフィールの「予約ボタン」として登録
- SNSのプロフィールに予約リンクを追加
- LINE公式アカウントのリッチメニューに設置

:::tip
初月は「ネット予約で初回来店の方は500円OFF」のような初回限定特典を出すと、従来の電話予約からの移行がスムーズです。
:::

## ❓ よくあるつまずき

| 症状 | 原因 | 対処 |
|---|---|---|
| 予約ページに空き枠が表示されない | スタッフに勤務時間が未登録 | Step 5 に戻り勤務時間を追加 |
| 希望日時を選んでも次に進めない | メニュー所要時間より短い枠を選択 | 長めの枠を選ぶ or 所要時間を調整 |
| 予約通知メールが届かない | 迷惑メールフィルタ | `@fujimin-pass.com` を受信許可リストに追加 |
| LINE予約と電話予約がぶつかる | 電話予約を管理画面に入力していない | スタッフが電話受付時、同じ画面で予約登録する運用に |

## 🎉 完了後にすること

- **1週目**: 予約件数・ネット経由比率を確認（管理画面 → ダッシュボード）
- **2週目**: リピーター率・無断キャンセル率を確認
- **1ヶ月後**: 売上が集計されるので、人気メニュー・曜日別の傾向を分析
