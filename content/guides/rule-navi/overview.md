## Rule Navi とは

**就業規則・マニュアル・社内ルールを PDF や Word のまま AI に学習させ、従業員が自然言語で質問できるようにするシステム**です。

「有給って何日取れるの？」「深夜残業の手当は？」といった質問に、該当規則の該当ページを引用しながら AI が即答します。

## こんな組織に向いています

| 当てはまる悩み | Rule Navi の機能 |
|---|---|
| 同じ質問が毎週来て答える手間が大変 | AI が規則の該当箇所を要約して回答 |
| 100ページの就業規則を誰も読まない | キーワードで該当部分をピンポイント表示 |
| 規則改定の周知が難しい | 新版をアップロードするだけで全員に反映 |
| 新入社員オンボーディングに時間が取られる | 自分で調べて解決できる環境 |
| 部署ごとに違うマニュアルが点在 | カテゴリで整理、部署別権限管理 |

## 主な機能

- **ドキュメント学習**: PDF / Word / Markdown をアップロードしてAIが学習
- **AI検索**: 自然言語で質問 → 該当ページを引用して回答
- **出典リンク**: 回答に「どの文書の何ページ」を明示
- **カテゴリ/タグ**: 部署別・テーマ別に整理
- **メンバー管理**: 招待・権限設定（一般／管理者）
- **利用ログ**: 誰が何を検索したかの分析（ハラスメント相談等の早期発見）
- **バージョン管理**: 古い版と新しい版の差分表示
