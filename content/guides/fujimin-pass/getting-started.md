## FUJIMIN PASS とは？

FUJIMIN PASS は「**店舗運営に必要なサービスを1つのIDでまとめて使える入口**」です。

1つのアカウントで:

- **Reserve Navi**: LINE予約管理
- **Ask Navi**: LINE AI自動応答
- **Shift Navi**: AIシフト管理
- **Rule Navi**: 社内規則AI検索
- **Social Navi**: SNS一元管理
- **ForProject**: プロジェクト管理
- （今後も追加予定）

がすべて使えます。必要なサービスだけ **オン** にすればOK。

## このガイドを読むとこうなります

| Before | After |
|---|---|
| サービスごとに別アカウントで混乱 | 1つのログインで全サービス管理 |
| 予約・シフト・SNSを別々に開く毎日 | 1画面から必要なツールに飛べる |
| 試しに使ってみたいが何から始めればいいかわからない | トライアルでリスクなく試せる |

## 所要時間・レベル

- **時間**: 最短10分
- **レベル**: ★☆☆（メール認証だけできればOK）
- **必要なもの**: メールアドレス、店舗名、代表者名

---

## Step 1: 新規登録

1. [fujimin-pass.com/register](https://www.fujimin-pass.com/register) にアクセス
2. 以下を入力:

| 項目 | 例 |
|---|---|
| メールアドレス | owner@example.com |
| パスワード | 8文字以上、英数字記号混在推奨 |
| 代表者名 | 山田 太郎 |
| 店舗/事業者名 | カフェ・ひだまり |
| 電話番号 | 090-xxxx-xxxx |

3. 「利用規約に同意して登録」をクリック

:::info
クレジットカード情報はこの時点では **不要**。各サービスを有料プランにアップグレードする時に登録します。
:::

---

## Step 2: メール認証

1. 登録したメールアドレスに「【FUJIMIN PASS】メール認証のお願い」が届く
2. メール内のボタン（またはリンク）をクリック
3. 認証完了画面 → ログインページへ自動遷移

:::warning
メールが届かない場合は:

1. 迷惑メールフォルダを確認
2. `noreply@fujimin-pass.com` を受信許可リストに追加
3. それでも届かなければ登録メアドを打ち間違えている可能性 → 再登録
:::

---

## Step 3: ダッシュボードの見方

ログイン後のホーム画面です。主要パーツ:

| エリア | 内容 |
|---|---|
| **左メニュー** | 利用中のサービス一覧（初期は空） |
| **「サービスを追加」ボタン** | 使いたいサービスを選んで有効化 |
| **上部ヘッダー** | 店舗情報・請求状況・通知 |
| **設定（右上歯車）** | プロフィール・パスワード・2段階認証 |

---

## Step 4: 使うサービスを選ぶ

1. ダッシュボード → 「**サービスを追加**」
2. サービス一覧が表示される:

| サービス | こんな方に | 料金目安 |
|---|---|---|
| Reserve Navi | 予約管理を効率化したい店舗 | ¥980〜¥4,980/月 |
| Ask Navi | LINE問い合わせをAI自動応答 | 月額制 |
| Shift Navi | 従業員のシフト管理 | 月額制 |
| Rule Navi | 社内規則をAI検索 | 月額制 |
| Social Navi | SNS一元管理 | 月額制 |
| ForProject | 個人プロジェクト管理 | フリーミアム |

3. 使いたいサービスのカードで「始める」をクリック
4. 各サービスの初期設定ガイドに沿って進める（各ガイドページあり）

:::tip
**まず1つだけ試す** のがおすすめです。複数同時に開始すると学習コストが上がります。
1ヶ月運用してから次のサービスを追加するのが王道パターンです。
:::

---

## Step 5: セキュリティ強化（推奨）

本格運用前に、**2段階認証**を必ず有効にしてください。

### 手順

1. 右上の歯車アイコン → 「セキュリティ」
2. 「2段階認証を有効にする」
3. スマホに認証アプリ（Google Authenticator / Microsoft Authenticator）をインストール
4. QRコードを読み取る → 6桁コードを入力
5. バックアップコード（10個）が表示される → **必ず紙に書くか安全な場所に保存**

:::warning
バックアップコードは再発行できません。スマホを紛失した場合、このコードがないとログインできなくなります。写真を撮って別の場所に保存しておくことを推奨します。
:::

---

## Step 6: 店舗プロフィールの完成度を上げる

右上「設定」→「店舗情報」で以下を入力しておくと、各サービスで自動で使い回されます。

- 正式な住所
- 営業時間・定休日
- 公式LINE ID
- ロゴ画像（各サービスのページでも表示される）
- 業種（マッサージ/飲食/美容室 等）

- **登録だけなら無料**（クレカ登録不要）

- 年契約プランだと月割 15% 程度お得
