## FUJIMIN PASS とは

**店舗運営や組織運営に必要なSaaSを1つのアカウントでまとめて使える、統合プラットフォーム**です。

Reserve Navi / Shift Navi / Ask Navi / Rule Navi / ForProject 等を、1つのメールアドレスで登録・請求管理できます。

## こんな方に向いています

| 当てはまる課題 | FUJIMIN PASS の仕組み |
|---|---|
| サービスを増やすたびに別アカウント作成で混乱 | 1つのIDで全サービス使える |
| 請求書がサービス毎にバラバラで経理が大変 | 1枚の請求書に統合 |
| サービスを試したいが最初から有料契約は抵抗 | 各サービスに無料トライアルあり |
| 社内で使う人が増えた時の権限管理が面倒 | 統一されたメンバー・権限管理 |

## 主な機能

- **統合アカウント**: 1メールアドレスで全サービスログイン
- **テナント/メンバー管理**: 複数人での利用時の権限設定
- **統合請求**: 全サービスの利用料を1枚の請求書に
- **2段階認証（2FA）**: アカウントセキュリティ強化
- **SSO**: 各サービスへのシームレスな遷移
- **監査ログ**: 重要操作を記録、セキュリティ/コンプライアンスに対応
- **利用分析**: 全サービス横断のKPIダッシュボード

## 料金の考え方

- **FUJIMIN PASS の登録自体は無料**（クレカ不要）
- **各サービスを個別に** 無料トライアル → 本契約
- 使わないサービスは解約すれば次月から課金停止
- 複数サービス契約で **統合割引** あり（3サービス以上で10%引き等）
