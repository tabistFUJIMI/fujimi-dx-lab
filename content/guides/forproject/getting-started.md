## このガイドを読むとこうなります

| Before | After |
|---|---|
| タスクを付箋・メモ帳・スマホメモで管理してバラバラ | 1つのアプリにまとめてスマホでもPCでも見られる |
| 複数案件を掛け持ちして「何が終わってて何が残ってるか」が曖昧 | プロジェクトごとに進捗率・残タスクが一目で分かる |
| クライアントから「進捗どうですか？」と聞かれるたびに焦る | 共有リンクを送るだけで進捗を見てもらえる |

## 所要時間・レベル

- **時間**: 最短5分で使い始められます
- **レベル**: ★☆☆（スマホで何かアプリ使えればOK）
- **必要なもの**: メールアドレスのみ

---

## Step 1: 新規登録

1. [forproject.fujimin-pass.com](https://forproject.fujimin-pass.com) にアクセス
2. 「無料で始める」→ メールアドレスとパスワードを入力
3. 確認メール → ログイン

:::info
ForProject はフリーミアム。最初の1プロジェクトは **完全無料** で機能制限なく使えます。
2つ目以降は月¥500/プロジェクトから。
:::

---

## Step 2: ホーム画面を理解する

ログイン直後の画面構成:

| エリア | 用途 |
|---|---|
| **プロジェクト一覧** | 進行中のプロジェクトがカード形式で並ぶ |
| **右上 + ボタン** | 新規プロジェクト作成 |
| **左下メニュー** | ダッシュボード / カレンダー / 設定 |
| **検索バー（上部）** | タスク名・メモ・タグで横断検索 |

---

## Step 3: 最初のプロジェクトを作る

### 手順

1. 右上 **「+」** ボタン → 「新規プロジェクト」
2. 以下を入力:

| 項目 | 例 |
|---|---|
| プロジェクト名 | 山田様 HP制作 |
| 開始日 | 2026-04-20 |
| 期限 | 2026-05-31 |
| カラー | お好みで選択（一覧での識別に便利） |
| 説明（任意） | コーポレートサイト全面リニューアル。ページ数 5、予算 40万円 |

3. 「作成」

:::tip
プロジェクト名は「**誰の・何の案件か**」が一目で分かる名前にすると、後で増えても迷いません。「山田様 HP制作」のように顧客名を入れるのがおすすめ。
:::

---

## Step 4: タスクを追加する

ここが実際の運用のメインです。

### 手順

1. プロジェクトを開く → 「タスク追加」
2. タスクごとに以下を入力:

| 項目 | 例 |
|---|---|
| タスク名 | トップページのラフデザイン |
| 締切 | 2026-04-25 |
| 優先度 | 高 / 中 / 低 |
| ステータス | 未着手 / 進行中 / 完了 |
| メモ | 参考サイト3つ、配色は青と白 |

3. タスクを一度にまとめて登録するのがコツ

### おすすめのタスク分割

プロジェクトを小さく分けると進捗が見えやすくなります:

```
山田様 HP制作
├─ 1. ヒアリング・要件定義 (4/22〆)
├─ 2. ワイヤーフレーム作成 (4/28〆)
├─ 3. デザイン案3パターン提示 (5/5〆)
├─ 4. デザイン最終決定 (5/8〆)
├─ 5. HTML/CSS実装 (5/20〆)
├─ 6. 画像・コンテンツ流し込み (5/25〆)
├─ 7. テスト・修正 (5/28〆)
└─ 8. 納品・請求書発行 (5/31〆)
```

:::tip
**「1日で終わる粒度」** がベスト。大きすぎるタスク（例: 「実装」）は進捗が曖昧になり、達成感も得にくいです。
:::

---

## Step 5: 日々の運用

### 朝のルーティン（1分）

1. ダッシュボードで「**今日の締切**」を確認
2. 緊急・重要の順にタスクを並び替え
3. 優先順位の高いものから着手

### 完了したら

1. タスクをタップ → 「完了」
2. **達成グラフ**が伸びる → モチベーション維持

### 週末レビュー（10分）

1. 今週完了タスク・未完了タスクを一覧
2. 遅延しているタスクは翌週にスライド
3. 来週のタスクを **上位3つだけ** 明確にしておく

---

## Step 6: クライアントと共有する（任意）

### 進捗を見せる

1. プロジェクト → 右上「共有」
2. 「閲覧専用リンク」を発行
3. リンクをクライアントに送る → クライアントはログイン不要で進捗ページを見られる

:::info
共有リンクには **パスワードを設定可能**。案件名・金額等が見える場合はパスワード必須にしてください。
:::

---

## 🎉 完了後にすること

- **1週間続けたら**: どのタスクが毎回時間オーバーしているかを振り返る
- **1ヶ月続けたら**: 完了タスク総数・平均リードタイム等の分析がダッシュボードに表示される
- **月末**: 各プロジェクトの工数を見て、次の見積もり精度を上げる

## 次のステップ

- **テンプレート機能**（準備中）: 似た案件を何度も作る方向け
- **時間計測機能**（準備中）: タスクごとの実工数を記録
- **請求書連携**（準備中）: 完了プロジェクトから請求書自動生成

## フリープランでの制限と有料化の判断基準

| 機能 | フリー | 有料 |
|---|---|---|
| プロジェクト数 | 1 | 無制限 |
| タスク無制限 | ✅ | ✅ |
| 共有リンク | 1個まで | 無制限 |
| カレンダー表示 | ✅ | ✅ |
| AI機能（タスク分解提案等） | ❌ | ✅ |

### 有料化の目安

- 案件が **2件以上同時進行** しだしたら月¥500投資する価値大
- AI機能が欲しくなったら（タスク自動分解は時短効果が大）
