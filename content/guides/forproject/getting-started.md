## このガイドを読むとこうなります

| Before | After |
|---|---|
| 幹事を任されたが何から手を付けるか分からない | AIが「このイベントならまずこれ」とタスクを提案 |
| 日程調整・出欠・会費集めがLINEとExcelでバラバラ | 1つのプロジェクトページに集約 |
| 幹事ひとりに負担が集中 | ロール機能で PM/CORE/LIGHT に役割分担 |

## 所要時間・レベル

- **時間**: 最短5分（最初のプロジェクト作成まで）
- **レベル**: ★☆☆（LINEアプリが使えればOK）
- **必要なもの**: LINEアカウント

---

## Step 1: LINE で新規登録

1. [forproject.fujimin-pass.com](https://forproject.fujimin-pass.com) にアクセス
2. 「**LINEで始める**」ボタンをタップ
3. LINE認証画面で許可 → 自動でアカウント作成・ログイン完了

:::info
メールアドレス入力も不要。LINEのプロフィール情報で即スタートできます。
:::

:::success
**初回登録ボーナス 100pt** が自動付与されます（90日有効）。まずはこのボーナス分でAI機能を試せます。
:::

---

## Step 2: ホーム画面を理解する

ログイン後のホーム画面:

| エリア | 用途 |
|---|---|
| **プロジェクト一覧** | 進行中のイベント・案件がカード形式で並ぶ |
| **右上 +ボタン** | 新規プロジェクト作成 |
| **左下メニュー** | ダッシュボード / AI相談 / ポイント残高 / 設定 |
| **ポイント表示** | 現在の残高と有効期限 |

---

## Step 3: 最初のプロジェクトを作る

### 手順

1. 右上 **「+」** ボタン → 「新規プロジェクト」
2. 以下を入力:

| 項目 | 例（飲み会の場合） |
|---|---|
| プロジェクト名 | 部署忘年会 2026 |
| 種別 | 飲み会 / イベント / 結婚式 / その他 |
| 開催予定日 | 2026-12-20 |
| 参加予定人数 | 15名 |
| 予算（1人あたり） | ¥5,000 |
| 概要（任意） | 駅近で個室のある居酒屋を予定 |

3. 「作成」

:::tip
種別を選ぶと、AIがその種別に特化したタスクを後で提案してくれます。飲み会なら「お店選び→日程調整→出欠確認→精算」といった流れを自動生成。
:::

---

## Step 4: AIにタスクを提案してもらう

ここが ForProject の肝心な機能です。

### 手順

1. プロジェクト画面 → 「AI相談」
2. 「**何から始めればいい？**」と話しかける
3. AIが項目ごとにタスクリストを提案:

例（忘年会の場合）:
```
- 日程候補を3つ挙げる → メンバーに投票してもらう
- お店候補を5つピックアップ（駅近・個室・予算）
- 幹事補佐を1名選ぶ
- 参加者全員に概要を告知
- 出欠確認締切を設定
- お店予約
- 会費徴収方法を決める
- 当日の役割分担
```

4. 気に入ったタスクを「採用」→ プロジェクトに追加される

:::info
AI相談1回あたり数ポイント消費します。初回ボーナス100ptなら10-20回は相談可能。
:::

---

## Step 5: メンバーを招待する

1. プロジェクト画面 → 右上「メンバー」
2. 「招待リンクを作成」
3. LINE で参加者にリンクを配布
4. メンバーはリンクから LINEログイン → 自動で参加完了

### ロールを設定

メンバーごとに役割を指定:

| ロール | 権限 | 典型的な人 |
|---|---|---|
| **OWNER** | 全権限、削除まで可 | 幹事（自分） |
| **PM** | 編集・タスク追加 | 副幹事 |
| **CORE** | 担当タスク編集 | 協力メンバー |
| **LIGHT** | 閲覧・コメント | 参加者全員 |

:::tip
**OWNERを2名にする** と、自分が忙しい時に副幹事がカバーできます。
:::

---

## Step 6: 日々の運用

### 毎日1分のルーティン

1. ForProject ホーム → 「今日のタスク」確認
2. 完了したタスクにチェック
3. 新たに気づいたタスクがあれば追加

### 週1回のミーティング前

1. プロジェクトを開いて進捗確認
2. 遅延タスクを一覧
3. AIに「停滞しているタスク、どう進めたら？」と相談

---

## 🎉 完了後にすること

- **1週目**: メンバー全員にログインしてもらい、全員が最新情報を共有できる状態に
- **イベント1週間前**: AI に「当日の役割分担を提案して」と相談
- **イベント後**: 「振り返りメモ」を残す → 次回企画時にAIが参考にしてくれる

## ポイントが足りなくなったら

1. 「ポイント」→「追加購入」
2. スタート（¥500/500pt）/ たっぷり（¥980/1,200pt）/ プレミアム（¥1,980/3,000pt）から選ぶ
3. Stripe決済ですぐ反映

:::info
大規模イベントや継続的な案件なら「プレミアム」パック（¥1,980/3,000pt）が割安です。
:::
