## このガイドを読むとこうなります

| Before | After |
|---|---|
| アプリを入れたが3日で使わなくなる | 毎朝1分で確認する習慣が身につく |
| 大きなタスクが進まない | 小さく分解、毎日「1つ終わる達成感」 |
| 見積もりが甘くて納期ギリギリ | 過去工数データから精度の高い見積もり |

---

## 基本原則：タスクは「1日で終わる粒度」に分解

:::info
ここが最大のコツです。これだけ意識すれば、他は自然と回ります。
:::

### NG 例

```
タスク: HP制作
期限: 1ヶ月後
```

→ 進捗が見えず、達成感もゼロ。結局後回し。

### OK 例

```
プロジェクト: 山田様 HP制作
├─ ヒアリング・要件整理（4/22）
├─ サイトマップ作成（4/24）
├─ ワイヤーフレーム Top（4/25）
├─ ワイヤーフレーム 会社概要（4/26）
├─ ワイヤーフレーム サービス（4/27）
├─ デザイン案 Top（4/30）
├─ デザイン案 会社概要（5/1）
├─ デザイン案 サービス（5/2）
├─ コーディング 共通パーツ（5/5）
├─ コーディング Top（5/7）
├─ コーディング 会社概要（5/8）
├─ コーディング サービス（5/9）
├─ コンテンツ流し込み（5/12）
├─ レスポンシブ対応（5/14）
├─ 公開前チェックリスト（5/16）
└─ 納品・請求書発行（5/20）
```

→ 毎日1-2個完了 = 毎日達成感。進捗が見える。

### 分解のコツ

- **1タスクの目安は 2-4時間**
- 「設計する」「実装する」「考える」はNG → 具体的な成果物に
- 「〇〇を書き終える」「〇〇の画面を作る」「〇〇をレビューする」

---

## 朝のルーティン（1分）

起床後、コーヒーを淹れながら:

1. ForProject を開く → 「今日の締切」を見る
2. 今日やるタスクを3つに絞る
3. 優先順位を決める（最も重要なものを最初に）

:::tip
**「今日やる」タスクを3つ以上にすると続きません**。2-3個に絞るほうが完遂率が高い。
:::

---

## 日中の運用

### タスクを始める時

1. 該当タスクをタップ → 「開始」ボタン
2. メモ欄に「今から作業開始」と書く（任意）

### 中断する時

- タスクを途中で止める場合：「中断」ボタン、メモに進捗を書く
- 「続きから再開」で簡単に戻れる

### 完了したら

1. タスクをタップ → 「完了」
2. **達成グラフが伸びる** → モチベーション維持

---

## 夕方のルーティン（3分）

仕事終わり・帰宅前に:

1. 今日完了したタスクを確認（✅達成感）
2. **明日やる3つ** をあらかじめピックアップ
3. 未完了で翌日にスライドするタスクは「期限調整」

:::success
この夕方ルーティンが最重要です。翌朝「何やろうかな」と迷う時間がなくなり、朝の集中力が2倍になります。
:::

---

## 週次振り返り（日曜 10分）

1. 今週完了タスク数をチェック
2. 遅延したタスクの原因を考える:
   - 見積もりが甘かった？
   - 予想外の割り込みがあった？
   - モチベ低下があった？
3. 来週の計画を立てる（大まかでOK）

:::tip
遅延原因を **メモ欄に記録** しておくと、3ヶ月後の見積もり精度が劇的に上がります。
:::

## クライアントとの進捗共有

### 共有リンクの作り方

1. プロジェクト画面 → 右上「共有」
2. 「閲覧専用リンクを作成」
3. パスワード設定（推奨）
4. クライアントにリンク + パスワードをメールで送る

### 何が見えるか

- プロジェクト名、期限
- タスク一覧と完了/未完了
- 全体の進捗率（％）
- **コメント欄**（クライアントが質問できる）

### 非公開にすべき情報

以下は共有リンクで **見せない** よう個別設定可能:

- 見積金額・報酬
- 内部メモ
- 他クライアントの情報（無関係タスクは非表示）

## 3ヶ月続けるための小技

| Tip | 効果 |
|---|---|
| スマホのホーム画面にPWAアイコンを置く | 起動が早く、毎日触る |
| 毎朝同じ時間に開く（朝9時等） | 習慣化 |
| 完了したタスクは消さずに残す | 達成履歴がモチベーション維持に |
| 失敗も記録する | 次の見積もり精度UP |
| 月1回、過去の実工数を振り返る | 自分の「得意/苦手」が見える |

## よくある質問

**Q. スマホとPCで同期されますか？**
A. 自動同期します。PWAなのでオフライン編集もOK、次回オンライン時に同期されます。

**Q. データのエクスポートは？**
A. 設定 → 「データエクスポート」で全データをJSON/CSVでダウンロード可能。

**Q. 有料化のタイミングは？**
A. 2つ目の案件を同時進行する時が多いです。フリーでしばらく使って慣れてからで十分。
