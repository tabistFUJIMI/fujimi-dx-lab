## ForProject とは

**飲み会・イベント・結婚式・プロジェクトの幹事をAIがサポートするツール**です。

「何人集まる？」「日程どうする？」「会費いくら？」「誰に何を頼む？」といった幹事の悩みを、1つのアプリで整理できます。共同編集とAIアシストで、幹事ひとりに負担が集中しません。

## こんな方に向いています

| 当てはまる悩み | ForProject の機能 |
|---|---|
| 飲み会の日程調整が LINE とカレンダーで往復して疲れる | プロジェクト内で日程調整・出欠・会場候補を一元管理 |
| 会費の集金がバラバラで管理が大変 | 会費管理機能で誰がいくら払ったかが一目で分かる |
| 招待する人が多くて案内が漏れる | 招待リンクをLINEで配るだけ、全員の状況が見える |
| 結婚式やイベントの準備が複雑で何から手を付けるか迷う | AIに「何やるべき？」と聞くとタスク一覧を提案してくれる |
| 幹事ひとりに負担が集中する | ロール機能で PM / CORE / LIGHT と役割分担 |

## 主な機能（実装済み）

- **プロジェクト管理**: イベント・幹事案件ごとに目標・日程・会費を管理
- **メンバー招待**: LINE OAuth ログイン、ロール指定（OWNER / PM / CORE / LIGHT）
- **AI機能**: タスク提案、相談チャット、要件分析（Haiku最適化でコスト低）
- **ポイント消費**: AI利用量に応じて課金、事前パック購入制
- **会費管理**: 集金状況を見える化
- **プロモコード**: 団体向け特別料金の付与・期限管理
- **Stripe決済**: 会費集金の自動化
- **PWA対応**: スマホでネイティブアプリのような操作感

## 料金（ポイントパック制）

ForProject は **買い切りのポイントパック制** です。サブスクではないので、使いたい時だけ購入できます。AI機能を使うとポイントを消費する仕組みです。

| パック | 料金（税込） | ポイント | 有効期限 | 典型的な用途 |
|---|---|---|---|---|
| スタート | ¥500 | 500pt | 購入から90日 | 飲み会・小規模イベントの幹事 |
| たっぷり | ¥980 | 1,200pt | 購入から90日 | 中規模イベント、複数案件の並行 |
| プレミアム | ¥1,980 | 3,000pt | 購入から90日 | 結婚式・祝賀会等、本格企画 |

:::success
**初回登録で 100pt 無料プレゼント**（90日有効）。まずは登録してボーナス分で使い心地を試せます。
:::

:::info
何度でも追加購入できます。プロジェクト途中でポイント不足になっても、その場で購入→即再開OK。
:::
