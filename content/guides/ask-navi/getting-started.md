## このガイドを読むとこうなります

| Before | After |
|---|---|
| LINE返信に追われて接客が中断 | よくある質問はAIが即答、返信ゼロ |
| 営業時間の問い合わせが毎日5件 | ゼロになる（AI対応・履歴でグラフ確認OK） |
| 「駐車場は？」「キャンセル料は？」を手打ちで返信 | 一度登録すれば半永久的にAIが使い回す |

## 所要時間・レベル

- **時間**: 最短20分（LINE公式アカウント準備込みで40分）
- **レベル**: ★☆☆（PCの基本操作ができればOK）
- **必要なもの**: PC or スマホ、LINE公式アカウント（未作成の方は Step 0 から）

---

## Step 0: LINE公式アカウントを用意する

:::info
すでに LINE公式アカウントをお持ちの方は Step 1 へ進んでください。
:::

1. [LINE Official Account Manager](https://www.linebiz.com/jp/entry/) にアクセス
2. LINEアカウントでログイン
3. 「アカウント作成」→ 店舗名・業種などを入力
4. 「Messaging API」を有効化（FUJIMIN PASS 連携時に必要）

:::warning
未認証アカウント（グレーバッジ）でもAsk Naviは動きますが、友だち追加QRの拡散力が弱くなります。認証アカウント化は LINE 側で無料で申請できます。
:::

---

## Step 1: FUJIMIN PASS に登録する

1. [fujimin-pass.com/register](https://www.fujimin-pass.com/register) にアクセス
2. メールアドレス・パスワードを入力
3. 店舗名と代表者氏名を入力
4. 確認メールのリンクをクリック

:::success
登録完了後、FUJIMIN PASS管理画面にログインできれば成功です。
:::

---

## Step 2: Ask Navi を有効化する

1. FUJIMIN PASS管理画面の左メニューから「サービス一覧」を開く
2. **Ask Navi** のカード内「始める」ボタンをクリック
3. プラン選択画面で「スタンダード」または「フリートライアル」を選ぶ

:::info
初月は無料トライアルがあります。気軽に試してから本契約できます。
:::

---

## Step 3: LINE公式アカウントと連携する

ここがメインです。FUJIMIN PASS と LINE を繋ぎます。

### 手順

1. Ask Navi 管理画面 → 「設定」→「LINE連携」
2. LINE Developers Console（別タブで開く）で:
   - チャネル基本設定 → **チャネルアクセストークン（長期）** を発行してコピー
   - **Webhook URL** に Ask Navi 管理画面に表示されている URL をコピー&ペースト
   - 「Webhookの利用」を **オン**
   - 「応答メッセージ」を **オフ**（AIが代わりに答えるため）
3. Ask Navi 管理画面に戻り、チャネルアクセストークンを貼り付けて「保存」

:::success
「LINE接続テスト」ボタンを押して **接続OK** と表示されれば成功です。
:::

:::warning
「応答メッセージ」を**オン**のままだと、LINE公式側の定型文とAskNaviのAIが**両方返信**してしまい、お客様が混乱します。必ずオフに。
:::

---

## Step 4: よくある質問（FAQ）を登録する

ここが **AIの頭脳** になる部分です。最初は5〜10個だけでも十分効果があります。

### 手順

1. Ask Navi 管理画面 → 「FAQ管理」→「新規追加」
2. 質問パターンと回答を入力:

| 質問（例） | 回答（例） |
|---|---|
| 営業時間 | 10:00〜21:00（最終受付20:30）です。定休日は水曜日。 |
| 駐車場 | 店舗前に2台、隣接コインパーキングもご利用いただけます。 |
| 予約方法 | [こちら](https://...) から24時間ネット予約受付中です。 |
| キャンセル料 | 前日まで無料、当日は施術料の50%です。 |
| 子連れOK？ | 可能です。ベビーカーでご来店の際はスタッフにお声掛けください。 |

3. 各FAQに「キーワード」を3〜5個登録（AIが質問と結びつける目印）

:::tip
最初は「店舗の常連さんから実際によく聞かれる質問トップ5」を入れるのが一番効果的です。完璧を目指さず、運用しながら足していくのが王道です。
:::

---

## Step 5: 動作テスト

1. 自分のLINEから店舗の公式アカウントを友だち追加
2. テストメッセージを送る（例: 「営業時間は何時まで？」）
3. 数秒以内にAIが登録したFAQを元に返答する

:::success
期待通りの返答が返れば完成！ここまでくればAsk Naviは24時間稼働します。
:::

### ❓ うまくいかないとき

| 症状 | 原因 | 対処 |
|---|---|---|
| 何も返信が来ない | Webhook URL が未設定 | Step 3 に戻って URL 設定を確認 |
| 「わかりませんでした」と返る | FAQに該当するキーワードが登録なし | FAQ管理でキーワード追加 |
| LINE公式の定型文が先に返る | 応答メッセージがオン | Step 3 の 2 に戻ってオフに |

---

## 🎉 完了後にすること

- 管理画面の **分析タブ** で「AIが答えられなかった質問」を定期的にチェック
- 未回答質問を見てFAQに追加 → AIがどんどん賢くなる
- 1週間後には AI対応率 80% 以上が一般的です
