### FAQ

1. **本当に無料で使えますか？** → ShiftNaviは完全無料。ReserveNaviも無料プランあり。
2. **初期費用はかかりますか？** → ¥0。月額費用のみ。
3. **契約期間の縛りはありますか？** → なし。いつでも解約OK。
4. **ホットペッパーと併用できますか？** → はい。ホットペッパー経由はそのまま、LINE直接予約をReserveNaviで管理する併用が可能。
5. **ITに詳しくないのですが？** → 3分で設定完了。スマホだけで使えます。
6. **お客様にアプリのダウンロードは必要？** → 不要。LINEだけで完結。
7. **ShiftNaviが無料なのはなぜ？** → シフト管理は全店舗の基本機能。まず使ってもらい、価値を実感していただくため。
