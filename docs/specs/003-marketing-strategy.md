## 4. AI活用マーケティング施策

### AskNaviデモ体験（最重要）

- `/demo/asknavi` にサンプル店舗のAIデモを公開
- レート制限: 1IP 10回/日、1セッション5回まで
- Haiku固定、max_tokens: 300
- 月1000回の質問でもHaikuなら$0.5以下
- CTA: 「このAIをあなたのお店でも → 月額¥550から」

### 無料DX診断AI

- `/tools/dx-check` に5問の選択式診断
- 結果をAI生成（業種×回答に応じたおすすめサービス）
- メール入力で詳細レポート送信 → リード獲得
- API: $0.001/回（Haiku）

### 営業メールAIパーソナライズ

- auto-mailに追加実装
- 送信先のWebサイトをHaikuが読み取り → 課題仮説を1行追加
- 130通 × Haiku = 月$0.1以下
- 開封率・返信率2-3倍の期待

### ShiftNaviバイラル設計

- 共有シフト表URLに「Powered by FUJIMIN PASS」表示
- スタッフ→別の店舗オーナーへの口コミ導線
- 紹介特典: 紹介者にポイント付与

### X投稿AI下書き

```
毎週月曜にHaikuで1週間分（3投稿）を一括生成
→ 手動で修正・スケジュール投稿
投稿時間: 7:00-8:00（出勤前）or 21:00-22:00（閉店後）
ハッシュタグ: #整体院経営 #サロン開業 #業務効率化 #SaaS #個人開発
```
